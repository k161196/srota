# srota

## Agent Skills

This repo ships reusable agent skills in `skills/`. Install them into your coding agent with the [`skills` CLI](https://github.com/vercel-labs/skills).

### Install all skills

```bash
npx skills add k161196/srota
```

### Install a specific skill

```bash
npx skills add k161196/srota --skill grilling
```

### Available skills

| Skill | Description |
|-------|-------------|
| `grilling` | Interview the user relentlessly about a plan or design. Use when you want to stress-test a plan before building. |
